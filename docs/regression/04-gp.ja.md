# Gaussian Process (GP) 回帰

> 🌐 [English](04-gp.md) | **日本語**

> 関数そのものに事前分布を置く **ノンパラメトリック・ベイズ** 回帰。
> 不確実性 (95% CI) も予測ごとに返却。
> `Model.GP` モジュール (多入力 + ARD + 多出力対応)。
>
> 関連: [04-kernel.ja.md](04-kernel.ja.md) (固定カーネル) / [04-rff.ja.md](04-rff.ja.md) (大規模近似) /
> 理論: [docs/principles/gp.ja.md](../principles/gp.ja.md)

## 1. モデル

\[ f \sim \text{GP}(0,\ k(x, x'))\quad y_i = f(x_i) + \varepsilon_i,\ \varepsilon_i \sim N(0, \sigma_n^2) \]

事後予測:
\[ \mu(x_*) = k_*^\top (K_y)^{-1} y,\quad
   \sigma^2(x_*) = k(x_*, x_*) - k_*^\top (K_y)^{-1} k_* \]

## 2. API

```haskell
import Model.GP

-- カーネル
data Kernel = RBF | Matern52 | Periodic

-- ハイパラ
data GPParams = GPParams
  { gpLengthScale  :: Double
  , gpSignalVar    :: Double
  , gpNoiseVar     :: Double
  , gpPeriod       :: Double
  , gpLengthScales :: Maybe (LA.Vector Double)  -- ARD
  }

defaultGPParams    :: GPParams
initParamsFromData :: [Double] -> [Double] -> GPParams

data GPModel  = GPModel  { gpKernel :: Kernel, gpParams :: GPParams }
data GPResult = GPResult { gpTestX, gpMean, gpVar, gpLower, gpUpper :: [Double] }

-- 1D API
fitGP    :: GPModel -> [Double] -> [Double] -> [Double] -> GPResult
fitGPMulti :: ...                                      -- multi-output
optimizeGP :: Kernel -> [Double] -> [Double] -> GPParams -> GPParams

-- 多入力 API (X is n × p)
buildKernelMatrixMV :: Kernel -> GPParams -> Matrix Double -> Matrix Double -> Matrix Double
fitGPMV       :: GPModel -> Matrix Double -> Matrix Double -> Matrix Double -> (Matrix Double, Vector Double)
optimizeGPMV  :: Kernel -> Matrix Double -> Vector Double -> GPParams -> GPParams
```

## 3. ミニマル例 (1D)

```haskell
import qualified Numeric.LinearAlgebra as LA

let xs = [0, 0.1, 0.2, ..., 1.0]
    ys = [...]
    p0 = initParamsFromData xs ys
    pOpt = optimizeGP RBF xs ys p0           -- 周辺尤度最大化
    res = fitGP (GPModel RBF pOpt) xs ys [0, 0.05, ..., 1.0]
-- res.gpMean / gpLower / gpUpper を可視化
```

## 4. カーネルの選び方

| カーネル | 滑らかさ | 用途 |
|---|---|---|
| `RBF` (Squared exponential) | C∞ | デフォルト、最も滑らか |
| `Matern52` | C⁴ | 物理系、若干粗い関数 |
| `Periodic` | 周期的 | 季節性データ (要 `gpPeriod`) |

## 5. ARD (Automatic Relevance Determination)

`gpLengthScales = Just v` で per-dim length scale を学習。**入力次元の重要度を自動推定** する効果。

```haskell
let p0 = (initParamsFromData ...) { gpLengthScales = Just (LA.konst 1 p) }
    pOpt = optimizeGPMV kern xMat yVec p0
-- pOpt.gpLengthScales が次元ごとの length scale
```

## 6. 多入力 (`fitGPMV`)

```haskell
let xMat = LA.fromLists [[x1_1, x1_2, x1_3], ..., [xn_1, xn_2, xn_3]] :: LA.Matrix Double
    yVec = LA.fromList [y1, ..., yn]
    p0 = initParamsFromData (concat (LA.toLists xMat)) (LA.toList yVec)
    pOpt = optimizeGPMV RBF xMat yVec p0
    (means, vars) = fitGPMV (GPModel RBF pOpt) xMat (LA.asColumn yVec) xTest
```

距離行列は BLAS GEMM で組まれ、reshape も `O(n²p)` の Haskell ループ無し。
詳細: [04-kernel.ja.md](04-kernel.ja.md#6-多次元入力-grammatrixmv--kernelridgemv--nwregressionmv) のセクションも参照。

## 7. 大規模 n での代替

| n | 推奨 |
|---|---|
| ≤ 1000 | `fitGPMV` (exact GP) |
| 1000-10000 | [`fitRFF`](04-rff.ja.md) (Random Fourier Features 近似) |
| > 10000 | sparse GP (今後追加予定) |

## 8. 頑健 GP (外れ値に強い)

`Model.GPRobust` で StudentT / Cauchy 観測尤度。

```haskell
import Model.GPRobust
let fit = fitGPRobust RBF p0 (RStudentT 4.0 0.1) xs ys
```
